# NaLI Field Problem Research

NaLI is framed as wildlife field intelligence infrastructure for Indonesia. The problems below are operational patterns surfaced from conservation technology practice, biodiversity data standards, and field data tooling references such as SMART, CyberTracker, GBIF, Darwin Core, IUCN, NASA FIRMS, Global Forest Watch, BirdNET, PostGIS, and H3. No deployment statistics are invented here.

## Sources Consulted

- SMART conservation tools: https://smartconservationtools.org/
- CyberTracker field data collection: https://cybertracker.org/
- GBIF API and occurrence data: https://techdocs.gbif.org/en/openapi/
