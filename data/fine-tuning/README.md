# NaLI Fine-Tuning Data Workspace

This folder contains small, synthetic, consent-safe examples for shaping decision-grade NaLI outputs. It does not contain real private observations, exact sensitive coordinates, passwords, personal data, or copied full papers.

Fine-tuning is not for memorizing species facts. NaLI should use source-backed retrieval and provider bridges for facts. Fine-tuning, if ever approved, is only for output discipline: short, metric-based, actionable, uncertainty-aware, bilingual, source-grounded, and field-ready.

Required safety gates before any real fine-tune:

- Written consent or anonymization for field records.
- No exact CR/EN sensitive coordinates.
- No user PII.
- No copyrighted full paper ingestion without license review.
- No legal overclaim.
- No `verified` wording unless review status supports it.
